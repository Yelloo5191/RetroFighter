RetroFigher based on Retro Tekken but in a 2D Style

Controls

Player 1:
MOVE `J,L`
JUMP `I`
CROUCH `K`
LEFT PUNCH `LEFT`
RIGHT PUNCH `RIGHT`
LEFT KICK `UP`
RIGHT KICK `DOWN`

Player 2:
MOVE `A,D`
JUMP `W`
CROUCH `S`
LEFT PUNCH `V`
RIGHT PUNCH `N`
LEFT KICK `G`
RIGHT KICK `B`
